# ADK Multi-Agent Travel System

This project implements a distributed multi-agent system using Google ADK and Gemini.

## Architecture

- Security Broker Agent (LLM Guard)
- Flight Agent
- Hotel Agent
- Planner Agent (Orchestrator)

## Deployment

- Google Cloud Run
- Vertex AI (Gemini)

## Endpoints

POST /flight  
POST /hotel  
POST /planner
